# EsKay Portfolio

A premium, interactive UX Portfolio built with React, Vite, and Framer Motion.

## Features
- **Premium Light Theme**: Clean, high-contrast aesthetic with noise texture overlays.
- **Interactive Hero**: Reference-matched composition with parallax and dual-font typography.
- **Editorial Projects**: Split-screen case study showcase with cinematic transitions.
- **Magnetic Specialties**: Bento-grid layout with mouse-following interactive glow.
- **Smooth Navigation**: Custom cursor and high-performance scroll triggers.

## Tech Stack
- **React 18**
- **Vite**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **CSS3** (Modern variables & layout)

## Setup
1. `npm install`
2. `npm run dev`
